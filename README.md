<h1>🔐 Python Cybersecurity Toolkit</h1> 
<h2>📌 Overview </h2>

<h2>This repository contains a collection of Python-based security tools designed to demonstrate foundational concepts in both offensive and defensive cybersecurity.

The toolkit focuses on real-world skills such as network reconnaissance, system interaction, and password security, with cross-platform support for Windows and Linux environments.</h2>

<h1>🚀 Getting Started</h1>
<h2>Requirements</h2>
<h2>
    <ul>
        <li>Python 3.x</li>
        <li>Passlib</li>
    </ul>
</h2>

<h1>⚙️ Tools Included:</h1>
<h1>🌐 IP Scanner</h1>
<h2><ul>
    <li>Scans a user-defined range of IPv4 addresses</li>
    <li>Identifies active and inactive hosts using ICMP (ping)</li>
    <li>Uses multithreading for faster performance</li>
    <li>Cross-platform support (Windows/Linux)</li>
    </ul></h2>
<h2> Key Concepts:<ul>
    <li>Network reconnaissance</li>
    <li>Parallel execution (ThreadPoolExecutor)</li>
    <li>OS-aware command handling</li>
    </ul></h2>

<h1>🖧 MAC Address Tool</h1>
<h2>(Unfinished</h2>

    <li>Cross-platform support (Windows/Linux)</li>
    </ul></h2>
<h2> Key Concepts:<ul>
    <li>System-level networking commands</li>

<h1>🔑 Password Generator & Evaluator</h1>

<h2><ul>
    <li>Generates secure passwords based on user-defined criteria:
